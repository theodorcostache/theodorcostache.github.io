clear            - clears the terminal screen
    login            - use this to view encrypted content 
    logout           - remove decryption key from browser cache
    exit             - closes the terminal screen
    profile          - displays a picture of me and my contact data (requires login)
    skills           - displays a summary of my skills (requires login)
    experience       - displays a table of my previous roles (requires login)
    languages        - displays a chart of the languages I speak (requires login)
    tech             - displays the technologies I worked with in the past (requires login)

    Hint: You can open multiple terminals using the icon in the top left corner
